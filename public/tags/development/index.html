Ffzap

Recently, I had lots of videos to re-encode to save some storage space. Some googling later I found that there’s not really a good way to run multiple ffmpeg processes in parallel or even just do batch processing.

Everybody just provided batch and shell scripts. Having to copy & paste and maybe also debug them, I wasn’t really satisfied. Also, controlling the output as in filename, extension, directory structure etc. is in theory super easy, but requires to adjust the script everytime. And so the idea of ffzap was born. Initially called mtc (Multithreaded Transcoding CLI), I later decided upon ffzap to highlight the two most important aspects of my tool – using ffmpeg under the hood and zap as in fast or quick.

Remove Youtube Statement Banner

How to remove the Statement / Theme Banners from YouTube

After years of these super annoying YouTube banners, I hacked together a little script that removes them from the DOM upon finding one.

To run this, you need to install something like Tampermonkey (available for Firefox and Chrome) and create a new userscript with the following code:

// ==UserScript==
// @name         Fuck YouTube Theme banners
// @namespace    https://codef0x.dev
// @version      1.0
// @description  Removes YouTube Theme banners
// @author       Tobias "CodeF0x" Oettl
// @match        *://*.youtube.com/*
// @grant        none
// ==/UserScript==

(() => {
    // remove on initial page load
    removeBanner();

    const target = document.querySelector('#content');

    const config = { attributes: false, childList: true, subtree: true };

    const callback = (mutationList, observer) => {
        removeBanner();
    }

    const observer = new MutationObserver(callback);
    observer.observe(target, config);
})();

function removeBanner() {
    const banner = document.querySelector('ytd-statement-banner-renderer');
        if (banner) {
            banner.parentElement.removeChild(banner);
        }
}

If you found an issue or a way to make it not run a thousend times when you just hover your mouse about something, add a comment on GitHub. You can also add a comment if the script stopped working due to YouTube doing some breaking changes.
